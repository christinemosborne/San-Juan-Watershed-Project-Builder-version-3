    How to use this tool

    • Start by entering the project name, Chapter, and a short description of the issue or opportunity.
    • Select the project type that best fits the situation (you can change it later if needed).
    • Review the generated pathway: who to call first, which entities to bring in next, and likely funding combinations.
    • Use the cross-cutting sequence to align Tribal, county, state, and federal coordination in the right order.
    • Treat this as a starting framework, not a substitute for legal review, Tribal approval, or agency consultation.
